# Loop N Roll

Basic desktop starter built with Electrobun, React 19, Vite, OXC lint/format, and `tsgo`.

## Scripts

- `pnpm install` installs the project and local toolchain binaries.
- `pnpm run dev` builds the renderer and starts Electrobun in watch mode.
- `pnpm run dev:hmr` starts Vite HMR alongside Electrobun.
- `pnpm run build` builds the renderer and the desktop bundle.
- `pnpm run check` runs lint, format check, and `tsgo` type-checking.

If you later install Bun globally, the same scripts also work with `bun run ...`.
